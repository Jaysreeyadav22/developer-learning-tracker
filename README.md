# 🚀 Developer Learning Tracker

A Python project to track my programming learning journey.

## Features
- 📚 Track Python topics (Completed / In Progress / Pending)
- 💻 Log HackerRank problems solved daily
- 🗺️ Roadmap with deadlines and priorities
- 📊 Summary dashboard with completion stats

## OOP Concepts Covered
- ✅ Inheritance
- ✅ Polymorphism
- ✅ Encapsulation
- ✅ Abstraction
- ✅ Composition

## How to Run
```bash
python tracker.py
```

## Tech
- Python 3.14
